# Coffee Shop

## Descripción del proyecto

Este proyecto es una página web para una cafetería que presenta información sobre el establecimiento, su menú, reservas y ubicación. La página web está diseñada para ser completamente responsive y seguir las mejores prácticas de desarrollo web.

## Funcionalidad

- Página de inicio con información sobre la cafetería
- Sección de menú con diferentes categorías de productos
- Formulario de reservas
- Información de contacto y ubicación
- Diseño responsive que se adapta a diferentes dispositivos

## Tecnologías utilizadas

- HTML5
- CSS3
- Metodología BEM para la organización del CSS
- Flexbox para el diseño de layouts
- Git y GitHub para control de versiones

## Planes de mejora

- Implementar JavaScript para hacer el formulario de reservas funcional
- Agregar animaciones CSS para mejorar la experiencia del usuario
- Integrar un mapa interactivo para mostrar la ubicación
- Añadir un carrito de compras para pedidos en línea
